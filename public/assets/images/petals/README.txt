# Hình Ảnh Hiệu Ứng Rơi

Vui lòng thêm 2 file ảnh vào thư mục này để hiệu ứng hoạt động:

1. **`dao.png`** - Hình cánh hoa đào (màu hồng)
2. **`mai.png`** - Hình cánh hoa mai (màu vàng)

## Lưu ý:
- Ảnh nên là **PNG trong suốt** (transparent background).
- Kích thước nhỏ gọn (khoảng 50x50px hoặc 100x100px) để tối ưu hiệu năng.
- Bạn có thể dùng ảnh cánh hoa đơn hoặc nguyên bông hoa nhỏ.
